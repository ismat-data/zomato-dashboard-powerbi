# 🍽️ Zomato Sales & Orders Dashboard (Power BI)

## 📊 Project Overview

This project presents an interactive **Zomato Analytics Dashboard** built using **Microsoft Power BI**. The dashboard provides insights into sales performance, order trends, customer ratings and city-level distribution of restaurant data.

The goal of this dashboard is to analyze restaurant orders, sales values and ratings to help stakeholders make data-driven decisions.

---

## 🛠️ Tools & Technologies Used

* **Microsoft Power BI Desktop**
* Data Modeling (Relationships & Measures)
* DAX (Data Analysis Expressions)
* Interactive Visualizations
* Custom Icons & Theming

## 📈 Key Metrics (KPIs)

The dashboard tracks the following important measures:

* **Sale_Value**
* **TopN_Sale**
* **Order_Count**
* **Rating_Count**

These KPIs help monitor overall business performance and top-performing categories.

## 📊 Dashboard Features

### 1️⃣ Sales Performance Analysis

* Displays total sales value
* Highlights top-performing restaurant types
* Compares sales across categories

### 2️⃣ Order Trend Analysis

* Year-wise order tracking using line chart
* Total order count KPI display

### 3️⃣ Category Distribution

* Clustered bar charts showing restaurant type performance
* Ranking-based Top N sales visualization

### 4️⃣ City-Level Insights

* Order distribution across cities
* Performance comparison between different locations

### 5️⃣ Rating Insights

* Total rating count monitoring
* Category-wise rating comparison

### 6️⃣ Interactive Filtering

* Advanced slicer for dynamic filtering by:

  * Year
  * Restaurant Type
  * City
